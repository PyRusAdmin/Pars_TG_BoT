### Скачивание видео с YouTube

1. Данные для прокси прописываются в .env.
2. Во время скачивания видео в консоль выводится информация о скачивании, возможно, потребуется разрешение на скачивание
   видео по ссылке https://www.google.com/device и нужно будет ввести код, предоставленный программой, например
   QHP-GQL-CFRN, далее нажать enter.
3. После скачивания видео будет находиться в папке downloaded_videos.
